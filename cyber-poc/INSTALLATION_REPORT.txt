🆘 استكشاف الأخطاء
================================================================================

إذا واجهت مشاكل:

1. مشكلة التثبيت:
   rm -rf .venv
   make setup

2. مشكلة الاستيراد:
   source .venv/bin/activate
   pip install --force-reinstall <package_name>

3. مشكلة الأذونات:
   chmod +x recon/*.sh recon/*.py

4. مشكلة البيئة:
   which python  # تأكد أنك في البيئة الافتراضية
